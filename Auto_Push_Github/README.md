# 🚀 Интерактивный Авто-Пушер для Fedora KDE (Wayland)

Умный Bash-скрипт с графическим интерфейсом на базе `kdialog` для быстрой сортировки, документирования и отправки локальных файлов (фотографий с iPhone, документов, скриптов) в Git-репозиторий на GitHub.

## ✨ Основные фичи
* **Полная интерактивность:** Скрипт задает 3 вопроса перед отправкой, позволяя гибко управлять структурой папок на лету.
* **Автоматический `README.md`:** Сам документирует состав выгрузки, добавляет дату, время и ваши личные комментарии.
* **Защита от «самопуша»:** Скрипт полностью игнорирует сам себя на этапе коммита — ваш репозиторий остается чистым от служебного кода.
* **Нативная интеграция с KDE Plasma 6:** Работает через системные окна ввода и уведомлений (актуально для Wayland/X11).

---

## ⚙️ Как это работает (3 сценария перед пушем)

При запуске скрипт сканирует рабочую директорию и поочередно выводит графические окна:

1. **Шаг 1: Общая папка**
   * Если ввести имя (например, `iphone_photos`), скрипт соберет абсолютно все файлы текущего уровня в этот каталог.
   * Если оставить поле пустым, файлы останутся в корне или перейдут к Шагу 2.
2. **Шаг 2: Персональные папки** (появляется, если пропущен Шаг 1)
   * Предлагает разложить каждый файл в личную папку, названную по имени самого файла (без расширения). Отлично подходит для изоляции бэкапов.
3. **Шаг 3: Кастомный README**
   * Предлагает ввести текстовое описание выгрузки. Ваш комментарий автоматически запишется в сгенерированный файл `README.md` в виде красивой цитаты.
