### 📝 Комментарий автора:
> # Portable PDF OCR To Clipboard / Text

Портативный Bash-скрипт для операционных систем на базе Linux (оптимизирован для **Fedora 43 / Wayland / KDE**), который автоматически находит PDF-скриншоты (например, сделанные через расширение *FireShot*), распознает на них текст и таблицы, сохраняет результат в `.txt` файлы и копирует всё содержимое прямо в буфер обмена для быстрой вставки в ИИ (Gemini, Claude, ChatGPT).

Скрипт полностью **мультиязычный** и одновременно распознает **русский, польский и английский** языки без потери спецсимволов.

## 🚀 Особенности
* **Полная портативность:** Скрипт не привязан к конкретным путям. Вы можете закинуть его в любую папку, и он обработает файлы именно внутри неё.
* **Массовая обработка:** Если в папке находится несколько PDF-файлов, скрипт поочередно обработает каждый из них.
* **Автоматизация для ИИ:** Весь распознанный текст объединяется и падает в буфер обмена (`Ctrl+V`), а также дублируется в локальные `.txt` файлы.

## 🛠️ Требования и установка зависимостей

Для работы скрипта требуются системные утилиты для извлечения графики, движок OCR Tesseract и инструмент работы с буфером обмена под Wayland.
